# .github

Org-level **community health files** and the **GattyWorks organization profile**.

- [`profile/README.md`](profile/README.md) — the public profile shown at
  [github.com/gattyworks](https://github.com/gattyworks).
- `profile/assets/` — brand images used by the profile.

GitHub automatically renders `profile/README.md` on the organization page, so edit
that file to change what visitors see. Changes go through a PR (never push to `main`).
